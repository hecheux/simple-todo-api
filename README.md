# Simple Todo API

Простой backend проект на Python
___


## Функциональность

Планируется реализовать:
- создание задачи
- получение списка всех задач
- получение одной задачи по **ID**
- обновление задачи
- удаление задачи
- изменение статуса задачи
- базовую валидацию входных данных

## Стек

На текущем этапе используется:
- Python
- Git
- GitHub

Далее планируется добавить:
- framework для API (FastApi)
